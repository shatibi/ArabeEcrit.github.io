Les Cas Grammaticaux — الإعراب

الرَّفْعُ • النَّصْبُ • الجَرُّ
```

En arabe, la terminaison d'un mot change selon sa fonction dans la phrase. C'est ce qu'on appelle l'i'rab (الإعراب). Il y a 3 cas principaux :

Cas Nom arabe Voyelle finale (indéfini) Voyelle finale (défini) Rôle principal
Nominatif رَفْع ـٌ (tanwin damm) ـُ (damm) Sujet (mubtada', fa'il)
Accusatif نَصْب ـً (tanwin fath) ـَ (fatha) Complément d'objet, khabar de kana
Génitif جَرّ ـٍ (tanwin kasr) ـِ (kasra) Après préposition, 2e mot d'une idafa
⚙️ Règle essentielle — Accord du na't

L'adjectif (na't) s'accorde en genre, en nombre, en définitude ET en cas avec le nom qu'il qualifie. Si le nom est au génitif, le na't l'est aussi.

📌 Exemple concret
‎ جَاءَ الطَّالِبُ الْمُجْتَهِدُ ← sujet = رفع → ـُ
‎ رَأَيْتُ الطَّالِبَ الْمُجْتَهِدَ ← COD = نصب → ـَ
‎ سَلَّمْتُ عَلَى الطَّالِبِ الْمُجْتَهِدِ ← après prép. = جرّ → ـِ

Le même mot طالب change de terminaison selon sa position.

💡 Astuce mnémotechnique : Pensez à "U — A — I" (damm → fatha → kasra) comme un escalier qui descend : sujet → objet → après préposition.
```
2

La Phrase Nominale — الجملة الاسمية

المُبْتَدَأ وَ الخَبَر
```
📐 Structure

Mubtada' (مبتدأ) = sujet nominal, toujours au nominatif (ـُ / ـٌ)
Khabar (خبر) = prédicat (ce qu'on dit du sujet), toujours au nominatif (ـُ / ـٌ)
Pas de verbe "être" au présent en arabe !

Le khabar peut être :

  • Un nom : مُحَمَّدٌ طَالِبٌ — Muhammad est un étudiant
  • Un adjectif : الْبَيْتُ كَبِيرٌ — La maison est grande
  • Un syntagme prépositionnel : الْكِتَابُ عَلَى الطَّاوِلَةِ — Le livre est sur la table
  • Une phrase verbale : الطَّالِبُ يَدْرُسُ — L'étudiant étudie
🔑 Règle d'accord Mubtada'/Khabar

Si le mubtada' est défini (avec ال), le khabar est généralement indéfini (tanwin) pour éviter une phrase équivoque.
الْوَلَدُ كَرِيمٌ ✓ — mais الْوَلَدُ الْكَرِيمُ = syntagme nominal (pas une phrase).

```
3

La Phrase Verbale — الجملة الفعلية

الفِعْلُ • الفَاعِلُ • المَفْعُولُ بِهِ
```
📐 Structure

Ordre de base : Verbe → Sujet (fa'il) → Complément (maf'ul)
فِعْلٌ + فَاعِلٌ + مَفْعُولٌ بِهِ

Élément Nom arabe Cas / Voyelle Exemple
Verbe فِعْل — كَتَبَ
Sujet (agent) فَاعِل رفع → ـُ / ـٌ الطَّالِبُ
COD مَفْعُول بِهِ نصب → ـَ / ـً الدَّرْسَ
⚠️ Accord du verbe en genre

Quand la phrase verbale commence par le verbe, celui-ci s'accorde uniquement en genre (pas en nombre) avec le sujet :
كَتَبَ الطُّلَّابُ (les étudiants ont écrit — verbe au singulier masculin)
كَتَبَتِ الطَّالِبَاتُ (les étudiantes ont écrit — verbe au singulier féminin)

✅ Exemple complet
‎ كَتَبَ الأُسْتَاذُ الدَّرْسَ عَلَى السَّبُّورَةِ

كَتَبَ = il a écrit (verbe, passé) | الأُسْتَاذُ = le professeur (fa'il, رفع) | الدَّرْسَ = la leçon (maf'ul, نصب) | عَلَى السَّبُّورَةِ = sur le tableau (prép. + nom جرّ)

```

La Conjugaison — Passé et Présent

الماضي وَ المضارع — كَتَبَ
```
🔤 Verbe de référence : كَتَبَ (kataba) — il a écrit

La racine est ك-ت-ب. Le schème du passé est فَعَلَ. Le schème du présent est يَفْعُلُ.

AU PASSÉ (الماضي) : on ajoute des suffixes à la racine de base كَتَبَ

Pronom Traduction Passé (ماضي) Suffixe ajouté
هُوَ il كَتَبَ — (base)
هِيَ elle كَتَبَتْ ـَتْ
هُمَا (m) ils deux (m) كَتَبَا ـَا
هُمَا (f) elles deux كَتَبَتَا ـَتَا
هُمْ ils كَتَبُوا ـُوا
هُنَّ elles كَتَبْنَ ـْنَ
أَنْتَ tu (m) كَتَبْتَ ـْتَ
أَنْتِ tu (f) كَتَبْتِ ـْتِ
أَنْتُمَا vous deux كَتَبْتُمَا ـْتُمَا
أَنْتُمْ vous (m) كَتَبْتُمْ ـْتُمْ
أَنْتُنَّ vous (f) كَتَبْتُنَّ ـْتُنَّ
أَنَا je كَتَبْتُ ـْتُ
نَحْنُ nous كَتَبْنَا ـْنَا

AU PRÉSENT (المضارع) : on ajoute des préfixes + des terminaisons à la racine

🔑 Schème du présent

La racine perd la voyelle initiale et reçoit :
→ un préfixe indiquant la personne (يَـ / تَـ / أَـ / نَـ)
→ une voyelle sur la 2e radicale (ici ـُـ pour كتب → يَكْتُبُ)
→ une terminaison indiquant le nombre/genre

Pronom Traduction Présent (مضارع) Préfixe + terminaison
هُوَ il يَكْتُبُ يَـ + ـُ
هِيَ elle تَكْتُبُ تَـ + ـُ
هُمَا (m) ils deux يَكْتُبَانِ يَـ + ـَانِ
هُمَا (f) elles deux تَكْتُبَانِ تَـ + ـَانِ
هُمْ ils يَكْتُبُونَ يَـ + ـُونَ
هُنَّ elles يَكْتُبْنَ يَـ + ـْنَ
أَنْتَ tu (m) تَكْتُبُ تَـ + ـُ
أَنْتِ tu (f) تَكْتُبِينَ تَـ + ـِينَ
أَنْتُمَا vous deux تَكْتُبَانِ تَـ + ـَانِ
أَنْتُمْ vous (m) تَكْتُبُونَ تَـ + ـُونَ
أَنْتُنَّ vous (f) تَكْتُبْنَ تَـ + ـْنَ
أَنَا je أَكْتُبُ أَـ + ـُ
نَحْنُ nous نَكْتُبُ نَـ + ـُ
💡 Mémorisez les 4 préfixes du présent : أ ن ي ت — "Je–Nous–Il/Ils–Tu/Elle". En arabe : أَنَيْتَ (anaytu) comme moyen mnémotechnique.
```

5

Les Pronoms Possessifs — الإضافة

الضَّمَائِرُ المُتَّصِلَة
```
📌 Principe de l'Idafa (الإضافة)

Pour exprimer la possession, on utilise soit :
1. Un pronom suffixe attaché au nom
2. Une construction génitive : nom au construct + nom au génitif

Le 1er nom d'une idafa perd son tanwin et ne prend pas de ال.

Personne Suffixe Exemple avec كِتَاب Traduction
هُوَ (il) ـهُ كِتَابُهُ son livre (à lui)
هِيَ (elle) ـهَا كِتَابُهَا son livre (à elle)
هُمَا (m/f) ـهُمَا كِتَابُهُمَا leur livre (à eux deux)
هُمْ (ils) ـهُمْ كِتَابُهُمْ leur livre (à eux)
هُنَّ (elles) ـهُنَّ كِتَابُهُنَّ leur livre (à elles)
أَنْتَ (tu m) ـكَ كِتَابُكَ ton livre
أَنْتِ (tu f) ـكِ كِتَابُكِ ton livre
أَنْتُمَا ـكُمَا كِتَابُكُمَا votre livre (à vous deux)
أَنْتُمْ (vous m) ـكُمْ كِتَابُكُمْ votre livre
أَنْتُنَّ (vous f) ـكُنَّ كِتَابُكُنَّ votre livre
أَنَا (je) ـِي / ـيَ كِتَابِي mon livre
نَحْنُ (nous) ـنَا كِتَابُنَا notre livre
⚠️ Cas particulier : voyelle avant ـِي

Quand on ajoute ـِي à un nom au nominatif, la terminaison ـُ devient ـِ :
كِتَابٌ → كِتَابِي (et non كِتَابُي)
La kasra s'impose avant le yaa' de la 1ère personne.
```

6

Le Na't — النعت (L'Adjectif Qualificatif)

النَّعْتُ وَ المَنْعُوت
```
📐 4 accords obligatoires du Na't

Le na't (adjectif) s'accorde avec le man'ut (nom) en :

  1. Genre — masculin / féminin
  2. Nombre — singulier / duel / pluriel
  3. Définitude — défini (ال) / indéfini (tanwin)
  4. Cas — رفع / نصب / جرّ
Phrase Na't (adjectif) Accord
طَالِبٌ مُجْتَهِدٌ مُجْتَهِدٌ masc. sg. indéf. رفع
طَالِبَةٌ مُجْتَهِدَةٌ مُجْتَهِدَةٌ fém. sg. indéf. رفع
الطَّالِبُ الْمُجْتَهِدُ الْمُجْتَهِدُ masc. sg. déf. رفع
الطَّالِبَانِ الْمُجْتَهِدَانِ الْمُجْتَهِدَانِ masc. duel déf. رفع
طُلَّابٌ مُجْتَهِدُونَ مُجْتَهِدُونَ masc. pl. indéf. رفع
رَأَيْتُ طَالِبًا مُجْتَهِدًا مُجْتَهِدًا masc. sg. indéf. نصب
```

Exercices Structurés

تَمَارِين
```
Exercice A — Cas grammaticaux

Mets la bonne terminaison (ـُ / ـَ / ـِ) sur le mot entre parenthèses :

  • 1جَاءَ (الطَّالِب) إلى الْفَصْلِ ← fa'il → cas ?
  • 2رَأَيْتُ (الأُسْتَاذ) في الْمَسْجِدِ ← maf'ul → cas ?
  • 3ذَهَبْتُ إلى (الْبَيْت) الْكَبِيرِ ← après prép. → cas ?
  • 4الْكِتَابُ عَلَى (الطَّاوِل) ← dans syntagme prép. → cas ?
  • 5قَرَأَ (الْمُعَلِّم) الدَّرْسَ ← fa'il → cas ?
  • 6سَمِعْتُ (الأَذَان) مِن الْمَسْجِدِ ← maf'ul → cas ?
✅ Corrigé

1. الطَّالِبُ (رفع) | 2. الأُسْتَاذَ (نصب) | 3. الْبَيْتِ (جرّ) | 4. الطَّاوِلَةِ (جرّ) | 5. الْمُعَلِّمُ (رفع) | 6. الأَذَانَ (نصب)

Exercice B — Conjugaison

Conjugue le verbe ذَهَبَ (aller) — même schème que كَتَبَ — pour les pronoms suivants :

  • 1هِيَ (passé)
  • 2أَنَا (passé)
  • 3هُمْ (passé)
  • 4أَنْتَ (présent)
  • 5نَحْنُ (présent)
  • 6هِيَ (présent)
  • 7هُمَا masc. (présent)
  • 8أَنْتُنَّ (passé)
✅ Corrigé

1. ذَهَبَتْ | 2. ذَهَبْتُ | 3. ذَهَبُوا | 4. تَذْهَبُ | 5. نَذْهَبُ | 6. تَذْهَبُ | 7. يَذْهَبَانِ | 8. ذَهَبْتُنَّ

Exercice C — Pronoms possessifs

Attache le bon suffixe possessif au mot بَيْت (maison) :

  • 1ma maison
  • 2ta maison (à toi, m)
  • 3sa maison (à lui)
  • 4sa maison (à elle)
  • 5notre maison
  • 6votre maison (pluriel m)
  • 7leur maison (pluriel m)
✅ Corrigé

1. بَيْتِي | 2. بَيْتُكَ | 3. بَيْتُهُ | 4. بَيْتُهَا | 5. بَيْتُنَا | 6. بَيْتُكُمْ | 7. بَيْتُهُمْ

Exercice D — Na't : Accord de l'adjectif

Accorde l'adjectif كَبِير avec le nom :

  • 1مَسْجِدٌ ........... (masc. sg. indéf. رفع)
  • 2مَدْرَسَةٌ ........... (fém. sg. indéf. رفع)
  • 3الْمَسْجِدُ ........... (masc. sg. déf. رفع)
  • 4رَأَيْتُ بَيْتًا ........... (masc. sg. indéf. نصب)
  • 5ذَهَبْتُ إلى مَدِينَةٍ ........... (fém. sg. indéf. جرّ)
  • 6مَسَاجِدُ ........... (masc. pl. indéf. رفع — pluriel brisé = fém. !)
✅ Corrigé

1. كَبِيرٌ | 2. كَبِيرَةٌ | 3. الْكَبِيرُ | 4. كَبِيرًا | 5. كَبِيرَةٍ | 6. كَبِيرَةٌ (pluriel non-humain → adjectif au féminin singulier !)

Exercice E — Construire des phrases

Construis une phrase nominale puis verbale à partir de ces éléments :

  • 1أُسْتَاذٌ / كَرِيمٌ → Phrase nominale
  • 2قَرَأَ / الطَّالِبُ / الْكِتَابَ → Phrase verbale
  • 3الْبِنْتُ / مُجْتَهِدَةٌ → Phrase nominale
  • 4شَرِبَتْ / الطَّالِبَةُ / الْمَاءَ → Phrase verbale
✅ Corrigé

1. الأُسْتَاذُ كَرِيمٌ | 2. قَرَأَ الطَّالِبُ الْكِتَابَ | 3. الْبِنْتُ مُجْتَهِدَةٌ | 4. شَرِبَتِ الطَّالِبَةُ الْمَاءَ
```

8

20 Phrases à Traduire — Pas à Pas

تَرْجَمَةٌ مُفَصَّلَة
```

Pour chaque phrase, je décompose la structure grammaticale avant de donner le résultat final en arabe.

1. Le professeur est dans sa classe.
TYPEPhrase nominale : mubtada' + khabar (syntagme prépositionnel)
SUJETالأُسْتَاذُ = le professeur → mubtada' → رفع → ـُ
PRÉPفِي = dans → حرف جرّ
POSS"sa classe" = فَصْلِهِ — فَصْل (classe) + ـهِ (son, à lui) → après prép. → جرّ → ـِ
الأُسْتَاذُ فِي فَصْلِهِ.
2. J'ai écrit la leçon sur le tableau.
TYPEPhrase verbale : verbe + fa'il (pronom inclus) + maf'ul
VERBEكَتَبْتُ = j'ai écrit (أنا) → suffixe ـْتُ
CODالدَّرْسَ = la leçon → maf'ul bih → نصب → ـَ
PRÉPعَلَى السَّبُّورَةِ = sur le tableau → عَلَى + السَّبُّورَةِ (جرّ)
كَتَبْتُ الدَّرْسَ عَلَى السَّبُّورَةِ.
3. Ton livre est grand.
TYPEPhrase nominale : mubtada' + khabar adjectif
SUJETكِتَابُكَ = ton livre → كِتَاب + ـكَ (suffixe "tu masc.") → mubtada' → رفع → ـُ (sur كتاب)
KHABARكَبِيرٌ = grand → khabar → رفع + indéfini → ـٌ
كِتَابُكَ كَبِيرٌ.
4. Les étudiantes sont intelligentes.
TYPEPhrase nominale au pluriel féminin
SUJETالطَّالِبَاتُ = les étudiantes → pl. fém. → mubtada' → رفع → ـُ
KHABARذَكِيَّاتٌ = intelligentes → pl. fém. indéf. rafa' → ـٌ (s'accorde en genre et nombre)
الطَّالِبَاتُ ذَكِيَّاتٌ.
5. Le messager d'Allah est un bon modèle.
TYPEPhrase nominale avec idafa (construction génitive)
SUJETرَسُولُ اللهِ = le messager d'Allah → رَسُول (1er nom idafa → رفع → ـُ, sans tanwin) + اللهِ (génitif → ـِ)
KHABARأُسْوَةٌ حَسَنَةٌ = un beau modèle → na't حَسَنَةٌ s'accorde (fém. car أُسْوَة est féminin)
رَسُولُ اللهِ أُسْوَةٌ حَسَنَةٌ.
6. Les musulmans prient dans la grande mosquée.
TYPEPhrase verbale : verbe (pl. m.) + fa'il + syntagme prépositionnel
VERBEيُصَلِّي → mais fa'il est pluriel masculin humain, donc le verbe reste au singulier en début de phrase : يُصَلِّي
FA'ILالْمُسْلِمُونَ = les musulmans → pl. masc. déf. → رفع → ـُونَ
PRÉPفِي الْمَسْجِدِ الْكَبِيرِ → الْمَسْجِدِ (جرّ ـِ) + الْكَبِيرِ (na't, déf. جرّ ـِ)
يُصَلِّي الْمُسْلِمُونَ فِي الْمَسْجِدِ الْكَبِيرِ.
7. Notre maison est devant l'école.
TYPEPhrase nominale : mubtada' possessif + khabar adverbe de lieu
SUJETبَيْتُنَا = notre maison → بَيْت + نَا → mubtada' → رفع → ـُ
KHABARأَمَامَ الْمَدْرَسَةِ = devant l'école → أَمَامَ (adv. lieu) + الْمَدْرَسَةِ (génitif car après une nisbah)
بَيْتُنَا أَمَامَ الْمَدْرَسَةِ.
8. Tu as ouvert (m) la porte.
TYPEPhrase verbale — sujet inclus dans le verbe
VERBEفَتَحْتَ = tu as ouvert (أَنْتَ) → فَتَحَ + ـْتَ → passé 2e masc. sg.
CODالْبَابَ = la porte → maf'ul → نصب → ـَ
فَتَحْتَ الْبَابَ.
9. Leur père est un médecin honnête.
TYPEPhrase nominale : idafa + khabar avec na't
SUJETأَبُوهُمْ = leur père → أَب (irrégulier : أَبُو au nominatif) + هُمْ (suffixe "leur")
KHABARطَبِيبٌ أَمِينٌ = un médecin honnête → khabar indéf. رفع + na't accordé
أَبُوهُمْ طَبِيبٌ أَمِينٌ.
10. Nous étudions la langue arabe chaque jour.
TYPEPhrase verbale au présent + COD + complément de temps
VERBEنَدْرُسُ = nous étudions → نَـ (préfixe نحن) + درس (racine) + ـُ
CODاللُّغَةَ الْعَرَبِيَّةَ → اللُّغَةَ (مف., نصب ـَ) + الْعَرَبِيَّةَ (na't, déf. fém. نصب ـَ)
TEMPSكُلَّ يَوْمٍ = chaque jour → كُلَّ (نصب, مف. زمان) + يَوْمٍ (جرّ après idafa)
نَدْرُسُ اللُّغَةَ الْعَرَبِيَّةَ كُلَّ يَوْمٍ.
11. Votre imam est dans la capitale de la France.
TYPEPhrase nominale : sujet possessif + khabar prép.
SUJETإِمَامُكُمْ = votre imam → إِمَام + كُمْ → mubtada' رفع ـُ
KHABARفِي عَاصِمَةِ فَرَنْسَا = dans la capitale de France → idafa : عَاصِمَة (جرّ après في) + فَرَنْسَا (génitif, nom propre invariable)
إِمَامُكُمْ فِي عَاصِمَةِ فَرَنْسَا.
12. Elle a lu deux livres utiles.
TYPEPhrase verbale : verbe fém. + fa'il + maf'ul duel avec na't
VERBEقَرَأَتْ = elle a lu → قَرَأَ + ـَتْ (féminin passé)
CODكِتَابَيْنِ نَافِعَيْنِ = deux livres utiles → duel نصب → ـَيْنِ pour le nom ET le na't
قَرَأَتْ كِتَابَيْنِ نَافِعَيْنِ.
13. La mère de Muhammad est ingénieure.
TYPEPhrase nominale : sujet idafa + khabar
SUJETأُمُّ مُحَمَّدٍ = la mère de Muhammad → أُمّ (رفع ـُّ) + مُحَمَّدٍ (génitif ـٍ)
KHABARمُهَنْدِسَةٌ = ingénieure → fém. indéf. رفع ـٌ
أُمُّ مُحَمَّدٍ مُهَنْدِسَةٌ.
14. Je suis allé(e) à la mosquée ce matin.
TYPEPhrase verbale au passé + complément de lieu + temps
VERBEذَهَبْتُ = je suis allé(e) → ذَهَبَ + ـْتُ
LIEUإِلَى الْمَسْجِدِ = à la mosquée → إِلَى + الْمَسْجِدِ (جرّ ـِ)
TEMPSهَذَا الصَّبَاحَ = ce matin → ظرف زمان (نصب ـَ)
ذَهَبْتُ إِلَى الْمَسْجِدِ هَذَا الصَّبَاحَ.
15. Ces examens sont faciles.
TYPEPhrase nominale avec démonstratif (اسم الإشارة)
SUJETهَذِهِ الاِخْتِبَارَاتُ = ces examens → هَذِهِ (démonstratif, pl. fém.) + الاِخْتِبَارَاتُ (رفع ـُ)
KHABARسَهْلَةٌ = facile (fém. sg. — pl. brisé non-humain traité comme fém. sg.) → رفع ـٌ
هَذِهِ الاِخْتِبَارَاتُ سَهْلَةٌ.
16. Mon père lit le Coran dans sa chambre.
TYPEPhrase verbale au présent + fa'il + COD + lieu
VERBEيَقْرَأُ = il lit (هو) → يَـ + قَرَأَ (racine) → présent sg. masc.
FA'ILأَبِي = mon père → أَب (irrég.) + ـِي (suffixe 1ère pers.) → رفع
CODالْقُرْآنَ = le Coran → maf'ul → نصب ـَ
LIEUفِي غُرْفَتِهِ = dans sa chambre → فِي + غُرْفَةِ (جرّ) + هِ (suffixe "son")
يَقْرَأُ أَبِي الْقُرْآنَ فِي غُرْفَتِهِ.
17. Les deux élèves sont devant la classe.
TYPEPhrase nominale au duel
SUJETالتِّلْمِيذَانِ = les deux élèves (masc. duel) → mubtada' → رفع → ـَانِ
KHABARأَمَامَ الْفَصْلِ = devant la classe → أَمَامَ + الْفَصْلِ (جرّ ـِ)
التِّلْمِيذَانِ أَمَامَ الْفَصْلِ.
18. Le gagnant est content.
TYPEPhrase nominale simple
SUJETالْفَائِزُ = le gagnant → mubtada' → رفع ـُ
KHABARمَسْرُورٌ = content → khabar → رفع indéf. ـٌ
الْفَائِزُ مَسْرُورٌ.
19. Vous (pl. masc.) avez salué le vendeur.
TYPEPhrase verbale au passé — sujet dans le verbe
VERBEسَلَّمْتُمْ = vous avez salué → سَلَّمَ + ـْتُمْ (أنتم)
PRÉPعَلَى الْبَائِعِ = sur / au vendeur → عَلَى + الْبَائِعِ (جرّ ـِ) — سَلَّمَ عَلَى = saluer quelqu'un
سَلَّمْتُمْ عَلَى الْبَائِعِ.
20. L'étudiant diligent a écrit un article dans le cahier.
TYPEPhrase verbale — fa'il avec na't + COD + lieu
VERBEكَتَبَ = il a écrit → hو implicite → passé sg. masc.
FA'ILالطَّالِبُ الْمُجْتَهِدُ = l'étudiant diligent → fa'il رفع ـُ + na't (déf. masc. رفع ـُ)
CODمَقَالَةً = un article → maf'ul → نصب indéf. ـً
LIEUفِي الْكُرَّاسَةِ = dans le cahier → فِي + الْكُرَّاسَةِ (جرّ ـِ)
كَتَبَ الطَّالِبُ الْمُجْتَهِدُ مَقَالَةً فِي الْكُرَّاسَةِ.
```

```
📖 Le texte

Un court texte narratif de niveau Médine 1 :

الطَّالِبُ يَذْهَبُ إِلَى الْمَدْرَسَةِ كُلَّ يَوْمٍ. يَقْرَأُ هُوَ الدُّرُوسَ وَ يَكْتُبُ الْوَاجِبَاتِ. الأُسْتَاذُ كَرِيمٌ. كَتَبَ الأُسْتَاذُ دَرْسًا جَدِيدًا عَلَى السَّبُّورَةِ. بَيْتُ الطَّالِبِ قَرِيبٌ مِنَ الْمَدْرَسَةِ.
Mubtada' (sujet nom.)
Khabar (prédicat)
Fa'il (agent verbal)
Maf'ul (COD)
Majrour (après prép.)
Verbe

Traduction : L'étudiant va à l'école chaque jour. Il lit les leçons et écrit les devoirs. Le professeur est généreux. Le professeur a écrit une nouvelle leçon au tableau. La maison de l'étudiant est proche de l'école.

⚠️ Point clé sur l'idafa

Dans بَيْتُ الطَّالِبِ : le mot بَيْت perd son tanwin (il n'est plus ـٌ mais ـُ) et ne peut pas prendre ال. Il est déterminé par la suite grâce à l'annexion. C'est la règle fondamentale de l'idafa.
```
